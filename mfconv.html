MODERNIZED FUTHORC

TRANSLITERATION

While the other rune sets you'll find here are historic, this Futhorc alphabet is of my own creation for use in the Beyond Ragnarok series. There'll be an explanation for how to use them on the main Rune page once I'm done with it. This algorithm is slightly smarter than the others, but only slightly.

  • A → ᚫ
  • B → ᛒ
  • C → ᚳ
  • CH → ᚳ
  • D → ᛞ
  • E → ᛖ
  • F(v) → ᚠ
  • G → ᚷ
  • ~G → ᚸ
  • H → ᚻ
  • I → ᛁ
  • J → ᛤ
  • K → ᛣ
  • L → ᛚ
  • M → ᛗ
  • N → ᚾ
  • ~N → ᛝ
  • O → ᚩ
  • P → ᛈ
  • Q → ᛢ
  • R → ᚱ
  • S(z) → ᛋ
  • SH → ᛋᚳ
  • ~S → ᛥ
  • T → ᛏ
  • TH → ᚦ
  • U → ᚢ
  • W → ᚹ
  • X → ᛉ
  • Y → ᛡ
